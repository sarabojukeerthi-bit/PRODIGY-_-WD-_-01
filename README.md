Responsive Landing Page

This project is a responsive landing page developed as part of my Web Development Internship.

Features
Fixed navigation menu
Hover effects on menu items
Navigation bar style changes on scroll
Responsive design for different screen sizes
Smooth scrolling experience
Technologies Used
HTML
CSS
JavaScript
Task Objective

Create an interactive navigation menu that changes color or style when scrolled or when hovering over a menu item. The navigation menu remains fixed and visible on all pages.
